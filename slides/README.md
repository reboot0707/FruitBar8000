# FruitBar8000 reveal.js 投影片

## 播放方式

直接用瀏覽器開啟 `slides/index.html` 即可播放。

如果開發環境已經有 VS Code Live Server、IIS Express 或其他靜態網站伺服器，也可以用它開啟專案根目錄，再瀏覽 `/slides/`。

## 操作

- `←` / `→`：上一張、下一張
- `S`：開啟講者模式（含約 5 分鐘講稿提示）
- `F`：全螢幕
- `Esc`：投影片總覽

投影片透過 CDN 載入 reveal.js 與字型，播放時需要網路連線；專案截圖與資料表關聯圖則直接使用儲存庫內的既有檔案。
